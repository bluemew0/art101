Ashley Lu

Hi! I'm Ashley and I'm a 3rd year AGPM major at UCSC. I love to draw, mostly 2D art and so I am working diligently to be a part of the game industry as a game artist. I'm interested in both 2D illustrations and creating concept art.

So far, I have enjoyed learning code. As of now, I am familiar with Python and JavaScript (comparatively less so), and currently, I am learning HTML and CSS :) I have a lot of fun problem solving in code and thinking differently to implement what I envision.
